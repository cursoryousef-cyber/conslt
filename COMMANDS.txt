========================================
   iOS - رفع على GitHub
========================================

git init

git add .

git commit -m "Doctor Consultation App - Complete"

# أنشئ repository على github.com/new أولاً
# ثم استبدل YOUR_USERNAME بحسابك:

git remote add origin https://github.com/YOUR_USERNAME/doctor-consultation.git

git branch -M main

git push -u origin main
