📱 لبناء Android APK + iOS IPA

========================================
   الجزء 1: Android APK (10 دقائق)
========================================

في PowerShell، اكتب كل أمر لوحده:

1. cd F:\appconsult

2. flutter clean

3. flutter pub get

4. flutter build apk --release

⏱️ انتظر 5-10 دقائق...

✅ APK سيكون في:
   build\app\outputs\flutter-apk\app-release.apk

========================================
   الجزء 2: iOS IPA (عبر Codemagic)
========================================

خطوة 1: إصلاح ملفات iOS
---------------------------
في PowerShell:

flutter create --platforms=ios .

عند السؤال overwrite: اكتب a ثم Enter

خطوة 2: رفع على GitHub
---------------------------
1. اذهب لـ: https://github.com/new
2. اسم Repository: doctor-consultation
3. انقر Create repository
4. انسخ الرابط

في PowerShell:
git init
git add .
git commit -m "Doctor Consultation App"
git remote add origin الرابط_الذي_نسخته
git branch -M main
git push -u origin main

خطوة 3: بناء على Codemagic
---------------------------
1. اذهب لـ: https://codemagic.io/signup
2. سجل بـ GitHub
3. Add application
4. اختر doctor-consultation
5. iOS workflow
6. Code signing: Automatic
7. Connect Apple Developer
8. Start build
9. انتظر 15-20 دقيقة
10. نزّل IPA!
